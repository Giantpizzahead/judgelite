Submission Form

(Memory limit is currently 128 MB [144 for Java]. Time limit is 3 seconds for C++, 4 for Java, and 5 for Python.)

Sample Problem Statement
You like to sum numbers. Please write a program that sums numbers.
Note that the sum might not fit into an 32-bit integer!

Input:
First line - A single integer N, the # of numbers to sum. N will be reasonably small.
Second line - N space separated integers. Find the sum of these integers.

Output:
A single line containing a single integer: The sum of the N numbers.
